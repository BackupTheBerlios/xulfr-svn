XUL peut être chargé soit à partir de votre système local de fichiers, installé dans le chrome, soit chargé à partir d'un site Web distant. XUL chargé depuis une site distant est sujet à des restrictions de sécurité vous interdisant d'appeler des composants natifs ou lire des fichiers et préférences. Vous pourrez réaliser ces opérations seulement avec du code signé ou du code installé dans le chrome.

Pour utiliser du XUL distant, placez simplement le fichier XUL et ces scripts associés sur un site distant. Assurez vous que votre serveur Web est configuré pour transmettre les fichiers XUL avec le type mime application/vnd.mozilla.xul+xml. Si votre serveur n'est pas configuré correctement, le fichier XUL sera soit affiché en texte plein, soit proposé en téléchargement à l'utilisateur.
