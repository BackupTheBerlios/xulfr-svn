Voici le gabarit le plus simple et le plus courant utilisant une syntaxe complète dans laquelle une liste d'enfants est parcourue.

Voir   Voir la source de données

<vbox datasources="animals.rdf" ref="http://www.some-fictitious-zoo.com/mammals">
  <template>
    <rule>
      <conditions>
          <content uri="?uri"/>
          <member container="?uri" child="?animal"/>
          <triple subject="?animal"
                  predicate="http://www.some-fictitious-zoo.com/rdf#name"
                  object="?name"/>
      </conditions>
      <action>
        <label uri="?animal" value="?name"/>
      </action>
    </rule>
  </template>
</vbox>
