Cet exemple montre comment générer du contenu HTML à partir d'un gabarit XUL. L'élément textnode crée du texte avec le prédicat 'name'.

Voir   Voir la source de données

<html:div datasources="animals.rdf" ref="http://www.some-fictitious-zoo.com/mammals"
          xmlns:html="http://www.w3.org/1999/xhtml">
  <template>
    <html:p uri="rdf:*"><textnode value="rdf:http://www.some-fictitious-zoo.com/rdf#name"/></html:p>
  </template>
</html:div>
