Ce gabarit génère du texte provenant de deux prédicats. Le premier est un label pour le nom de famille et le second est un description pour la description de la personne. La balise textnode est utilisable uniquement dans un gabarit où elle crée une zone de description tronquable.

  <template>
    <vbox uri="rdf:*">
      <label class="header" value="rdf:http://www.some-fictitious-zoo.com/rdf#name"/>
      <description>
        <textnode value="rdf:http://www.some-fictitious-zoo.com/rdf#description"/>
      </description>
    </vbox>
  </template>
</vbox>
