Ce répertoire contient le projet de traduction de tutoriels XUL du site
http://www.xulplanet.com

Chaque tutoriel contient un script "build" pour générer les pages html du tutoriel, et
deux ou trois de ces répertoires :

* src/ : contient les sources des articles, c'est à dire les articles stockés en html;
mais ce ne sont pas de vrais pages HTML (pas d'en-tête). Ces fichiers ne contient que le corps
de l'article.
* builds/ : répertoire où sont stockées les pages générées par le script "build" . Il n'est pas
nécessaire d'enregistrer celles-ci dans le cvs.
* static/ : c'est le repertoire du tutoriel ne contenant que les fichiers statiques.

Procédure pour générer le tutoriel :

1. executer le script "build" en ligne de commande (son nom varie en fonction du tuto). Ex :
   # php buildtuto.php
2. mettre le contenu de static/ et celui de builds/ dans le même répertoire sur le site
   (xultu/ par exemple)
3. y mettre également *le contenu* de common/ (et pas common)
4. vous pouvez naviguer dans ce répertoire
